## ⚔️ L'Arsenal du Maître

Point de fioritures inutiles ici. Nous utilisons l'essentiel pour atteindre l'excellence :

* **Vite** : L'éclair qui illumine le développement et forge le build final avec rapidité.
* **Bootstrap & Icons** : La structure et le symbole, assurant que l'interface reste harmonieuse sur tous les supports.
* **Date-fns** : La maîtrise du temps, pour manipuler les cycles et les dates avec précision.
* **ESLint & Prettier** : La discipline du sabre, garantissant un code propre, droit et sans impureté.

## 📜 Voie de l'Installation

Pour faire revivre ce temple sur votre machine locale, suivez le sentier tracé :

1. **Invoquer le code**
```bash
git clone https://github.com/monamijer/portfolio.git
cd portfolio

```


2. **Éveiller les outils**
```bash
npm install

```


3. **Méditer en mode développement**
```bash
npm run dev

```


4. **Purifier et reconstruire** (Si l'esprit du code s'embrume)
```bash
npm run reinstall

```

🏗️ L'Équilibre de l'Édifice (Architecture)

Dans ce jardin de code, la structure suit la fonction. L'organisation est pensée pour que le flux d'informations circule sans entrave, comme l'eau dans un ruisseau de montagne.

    Le Cœur (main.js) : C'est le centre de gravité. C'est ici que l'énergie est canalisée, que les modules s'assemblent et que le premier souffle est donné à l'interface.

    La Forme (Bootstrap & Icons) : Nous ne réinventons pas la roue ; nous utilisons des piliers solides pour que le design soit à la fois stable et accueillant pour le visiteur, quel que soit son écran.

    Le Rythme (Date-fns) : Le temps est notre seul véritable luxe. Cette bibliothèque nous permet de l'organiser avec une élégance mathématique, transformant des données brutes en moments compréhensibles.

    La Clarté (Scripts de Forge) : Les commandes dev et build via Vite ne sont pas que des outils techniques ; ce sont des processus de raffinage qui transforment le métal brut de vos fichiers en une lame de cristal prête pour le web.

Cher visiteur, si vous prenez le temps de lire ces lignes, voici quelques pépites de savoir glanées sur le chemin de l'informatique :

    Le Mystère du "Bug" > Saviez-vous que le premier "bug" informatique était une véritable mite (un insecte) coincée dans un relais du calculateur Mark II en 1947 ? Parfois, la solution n'est pas dans la logique, mais simplement dans le nettoyage de ce qui nous encombre.

    L'Art de l'Invisible > Dans le code comme dans la vie, ce qui ne se voit pas est souvent le plus important. Un bon README est comme une lanterne dans la nuit : il ne marche pas à votre place, mais il éclaire le chemin pour que vous ne trébuchiez pas.

    La Magie du 404 > L'erreur 404 est devenue un symbole universel de l'égarement. Mais saviez-vous qu'elle tire son nom (selon la légende) du bureau 404 au CERN où se trouvaient les premiers serveurs du Web ? Chaque erreur est un rappel de nos origines.
