# 🚀 Mon Portfolio - [Monami Jerome]

Bienvenue dans le dépôt de mon portfolio personnel. Ce projet est une application web moderne, interactive et responsive, conçue pour présenter mes projets, mes compétences et mon parcours.

🔗 **Démo en ligne :** https://monamijer.github.io/portfolio/

---

## 🛠️ Technologies & Dépendances

Ce projet est bâti avec l'écosystème **React / Next.js** et utilise les bibliothèques suivantes pour l'interactivité et le style :

### Coeur du projet
- **Next.js (13.4.8)** : Framework React pour le rendu hybride.
- **TypeScript** : Pour un typage statique et un code robuste.
- **Sass & Tailwind CSS** : Pour un design moderne et utilitaire.

### Animations & 3D
- **Framer Motion** : Pour les animations fluides et les transitions de pages.
- **Three.js & @react-three/fiber** : Pour l'intégration d'éléments 3D.
- **@react-three/drei** : Helpers utiles pour les scènes Three.js.

### Utilitaires
- **React Icons** : Bibliothèque d'icônes.
- **React Vertical Timeline** : Pour la section parcours/expérience.
- **Resend & React Email** : Pour la gestion du formulaire de contact et l'envoi d'emails.

---

## 📦 Installation et Lancement

Pour cloner et lancer ce projet localement, suivez ces étapes :

1. **Cloner le dépôt**
   ```bash
   git clone [https://github.com/monamijer/portfolio.git](https://github.com/monamijer/portfolio.git)
   cd portfolio

```

2. **Installer les dépendances**
```bash
npm install

```


3. **Lancer le serveur de développement**
```bash
npm run dev

```


L'application sera disponible sur `http://localhost:3000`.
4. **Build pour la production**
```bash
npm run build

```



---

## 📂 Structure du projet

* `/components` : Composants React réutilisables.
* `/public` : Actifs statiques (images, modèles 3D).
* `/styles` : Fichiers globaux CSS/Sass.
